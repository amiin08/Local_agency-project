# Local_agency-project
Responsive local agency website built with modern frontend tools to showcase services and drive client engagement.This project is designed to help local businesses establish a strong online presenceand attract potential clients.

## Features
-Fully responsive design
-Clean and modern UI
-Services and portfolio sections
-Contact section for client inquiries 
-Fast loading and user-friendly layout

## Technologies Used
-HTML5
-CSS3
-JavaScript
